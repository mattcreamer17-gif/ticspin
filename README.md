# TicSpin Landing Page

This is a bespoke, single-page landing site prepared for TicSpin Market (booth SMB26, eMerge Americas 2026). It uses TicSpin's own brand tokens pulled from ticspin.com (Astra WordPress theme): Barlow Semi Condensed headings, Roboto body, the blue (#0170B9) link color over white with dark grey body text, subtle shadow, and a small topbar carrying the Spanish-language trust signal and same-day delivery line. The page carries the fit-analysis section 7 pitch reflowed into the storefront's short, direct voice, with a single primary CTA to Matt Creamer (CRO)'s 30-minute Calendly link. Eduba branding is confined to the footer. Pricing and revenue-share detail are stripped from the public-facing copy; NLP Logix is named with its mandated credibility stats; the ICM paper link is live. The page renders cleanly at 375px and scales up through a single 880px content column.
